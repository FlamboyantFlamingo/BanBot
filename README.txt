Commands:
(Prefix)kick @user reason
(Prefix)ban @user reason

Use for custom kick and ban messages:
Command: !kick (user) (reason) / !ban (user) (reason)
%reason% - the reason the user is being kicked/banned
%mod% - the person who is running the command

EXAMPLE:
You have been kicked by %mod% for: "%reason%". Please remember the rules before you rejoin.

Input - !kick @somebody you suck

User sees - You have been kicked by @ModeratorPerson for: "you suck". Please remember the rules before you rejoin
